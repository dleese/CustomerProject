## Description
This is the first Draft of the idea to automatically implement a Version/Revision control system for **Markdown** and **asciiDoc** files. The tool should add a table under the heading of **Record and Revisions** at the beginning of the document with the following information, once it has been modified from the previous Version/Revision.

- issue no.
- date of issue
- change 
- Author
